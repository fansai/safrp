# safrp cn/[en](#)
safrp（Simple and fast reverse proxy）是基于Go语言开发的一个轻量级和可定制化功能的内网穿透软件，支持 TCP、UDP、SSH、HTTP、HTTPS、WebSocket 协议的数据转发。
可根据场景在服务端或内网中进行限流、IP请求记录的内网穿透软件
### 功能
目前仅实现HTTP协议数据的转发
